# Timeline

Monthly timeline data showing daily activities and momentum scores.

## Source
- Endpoint: `/timeline?month=YYYY-MM`
- Last synced: 2026-03-31

## Contents
- `challenges.json` — 18 timeline challenges
- `YYYY-MM/YYYY-MM.json` — Monthly timeline data (27 months: 2024-01 to 2026-03)

## Schema (key fields)
- `items` — Array of timeline items per day
- `momentum_scores` — Daily momentum scores for the month
